# Day 1 - Descriptive Statistics with Python

## Overview
This project is part of my Machine Learning learning journey. In this notebook, I explored the fundamentals of descriptive statistics using Python by analyzing a student performance dataset.

## Topics Covered
- Creating and working with Pandas DataFrames
- Saving datasets as CSV files
- Calculating percentiles (Q1, Median, Q3)
- Correlation analysis
- Scatter plot visualization
- Probability calculations
- Histogram visualization
- Outlier detection using Z-Score
- Outlier detection using IQR
- Box plot visualization
- Summary statistics using `describe()`

## Libraries Used

- Matplotlib
- SciPy
